## idea常用快捷键
Ctrl＋E：显示最近编辑的文件列表  
Ctrl＋F12：显示当前文件的结构  
Ctrl＋P：显示方法的参数信息  
Ctrl＋Alt＋V ：自动创建变量也就是=号前的变量定义  
Ctrl+Alt+L：  格式化代码  
ALT+INSERT：生成get/set 实现接口等  
Alt+回车：导入包自动修正
